# arcflow-dashboard

This repository exists solely to host the **auto-generated Arcflow daily dashboard image**.

The image is regenerated and pushed daily from an external process — there is no
application code here. The latest dashboard is always available at:

```
https://gitlab.com/<namespace>/arcflow-dashboard/-/raw/main/dashboard.png
```

`dashboard.png` is the only meaningful file in this repo and is overwritten on each update.
